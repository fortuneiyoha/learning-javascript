# Lesson 6 - Spread Operator

The spread operator copies elements from an array into another.

#### Example

```js
const userName = ["Fortune", "O."];

// Spreads the elements from `userName` into `names`
const names = [...userName, "Iyoha"];

console.log(names); // Outputs: [ 'Fortune', 'O.', 'Iyoha' ]
```
